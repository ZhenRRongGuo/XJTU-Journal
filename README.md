# 使用须知
* 通过 \documentclass{XJTU-Journal} 使用，请将 .cls 文件放置在 .tex 相同的目录
* 使用 XeLaTeX 编译
* 本模板通过西安交通大学学报官网 Word 模板制作而成，如果您因为某个课程作业需要，但又不想用 Word，那么可以使用此模板
* !! 本模板仅为自用，不得用于学报投稿，如果需要投稿，请安装期刊需求提供稿件
* 参考文献使用 biblatex，由 biber 编译，因此编译顺序为 xelatex -> biber -> xelatex -> xelatex
* 部分编译器可能不支持中文名称的图片插入（\includegraphics{}），或不支持 .tex 文件中文命名，请您根据实际情况调整
* 建议您详细阅读 main.pdf 来了解本模板
